📋 Technical Details (click to expand)
How it works:
  • Generates all required branding assets using HTML5 Canvas API
  • Optimizes file sizes to meet Microsoft's strict limits (PNG: 50KB, JPEG: 300KB)
  • Creates separate light and dark theme variants
  • Packages everything with a PowerShell deployment script
Generated Assets:
  • Banner Logo (280×60px): Used in email footers and some app headers
  • Header Logo (245×36px): Displayed in app headers and navigation
  • Square Logos (240×240px): Light/dark variants for app tiles
  • Favicon (32×32px): Browser tab icon
  • Backgrounds (1920×1080px): Sign-in page backgrounds for both themes
PowerShell Script Features:
  • Uses Microsoft Graph PowerShell SDK
  • Validates file sizes before upload
  • Configures both default (light) and en-US (dark) localizations
  • Handles errors gracefully with retry guidance

Branding Configuration

Image Requirements (Microsoft Specs):

  • • Sign-in background: 1920×1080px, JPEG, max 300KB
  • • Banner logo: 280×60px, PNG, max 50KB
  • • Header logo: 245×36px, PNG, max 10KB
  • • Square logos: 240×240px, PNG, max 50KB
  • • Favicon: 32×32px, PNG, max 5KB
